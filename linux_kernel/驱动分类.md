# 1. 存储设备驱动

```mermaid

graph TD;
    A[存储设备驱动] --> B[字符设备驱动];
    A --> C[块设备驱动];
    A --> M[SCSI设备驱动];

    B --> B1[用途: 直接与硬件进行字节流交互的设备];
    B --> B2[示例: 键盘、鼠标、串口、终端设备等];

    C --> C1[用途: 存储介质的设备];
    C --> C2[示例: 硬盘、固态硬盘、闪存等];

    M --> M1[用途: SCSI总线上的设备];
    M --> M2[示例: 磁盘阵列、光驱等];


```

# 3. 总线和桥接设备驱动
```mermaid

graph TD;
    A[总线和桥接设备驱动] --> G[I2C设备驱动];
    A --> H[SPI设备驱动];
    A --> I[平台设备驱动];
    A --> R[桥接设备驱动];
    A --> S[总线驱动];

    G --> G1[用途: I2C总线上的设备];
    G --> G2[示例: 传感器、EEPROM等];

    H --> H1[用途: SPI总线上的设备];
    H --> H2[示例: 传感器、显示屏控制器等];

    I --> I1[用途: 嵌入式系统中的特定硬件平台];
    I --> I2[示例: ARM架构中的各种外设];

    R --> R1[用途: 桥接不同总线或设备之间的通信];
    R --> R2[示例: PCI桥、USB桥等];

    S --> S1[用途: 管理总线设备];
    S --> S2[示例: PCI总线驱动、USB总线驱动、I2C总线驱动等];

```
